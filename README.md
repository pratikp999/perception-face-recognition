# perception-face-recognition

Real-Time Face Perception Pipeline
Overview
A computer vision pipeline designed for high-accuracy face detection and recognition. This project explores the balance between model depth and real-time inference speed, critical for robotic perception.

Technical Highlights
Framework: Python, OpenCV, YOLOv8/v11.

Optimization: Implemented frame-skipping and resolution scaling to maintain 30+ FPS on standard hardware.

Data Handling: Custom scripts for image augmentation and dataset balancing.
